Browser Versions

There will be two different designs used for this website. This is to handle anyone using Internet Explorer before version 9. We are currently showing the IE compatible version.

Home Page and Navigation

The home page is located at https://timoun-production.appspot.com. On this page, you will see a welcome message and a brief overview of Timoun.

The Navigation bar has several links, each which lead to a separate page.

About

The about page is meant to give the public more information about Timoun.

Search

This page will allow for users to search for services in the Timoun database. It will also explain how best to search the database. How to use filters, keywords etc to narrow down their search.

Contact IBESR

This page will explain a little bit about IBESR(Institut du Bien-Etre Social et de recherches), and give contact information for that organization.

Suggest Services

This page will allow the public to suggest new community services.

Mental Health Services

This page will serve as a directory of mental health services in the database.

Login/Logout

This page will allow users to login or logout. This process will be described in more detail below

Users and Login/Logout

Add a User

Users are added by an admin.

An Admin will click Users on the navigation bar. After that, the admin will click the Add User button.

After the admin fills out the form, the user will get a confirmation email, which will explain how they can set their password.

View and Delete Users

An Admin can click Users on the navigation bar to see all users. On that page, they can see basic user information, and can click on the email to edit or delete that user.

Forgot Password

If a user forgets their password, they can click Login and click Forgot Password. Clicking either of these links will log you out. So you'll have to log back in if you use them.

From here, a user will enter their email, and receive and email link explaining how they can reset their password safely.

Dashboard and Audits

View Audits on the Dashboard

Timoun keeps track of all changes to the database.

An admin can see all changes by clicking Dashboard on the navigation panel.
